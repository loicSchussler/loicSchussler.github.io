Moodle like

Github
Etat : fini
Contexte : scolaire (Dorset College, Dublin)
Projet réalisé en groupe de 4.

Plateforme internet programmé en C# copiant le principe de moodle : permettre la gestion des notes,devoirs,frais de scolarités des élèves ainsi que l'organisation d'une université.

Outils informatiques utilisés

  • Langage C# avec le framework ASP.Net Core MVC
  • Visual Studio 2019 en tant qu'environnement de développement.
  • MySQL Workbench pour gérer la base de données.
  • Git pour collaborer et mettre à jour le code du projet.
  • Discord pour communiquer au sein de l'équipe.
  • Trello pour se répartir les tâches.

Cahier des charges du projet

Objectif global

La plateforme doit être composée de 3 branches (étudiants,professeurs et membres de l'administration) et doit permettre d'inscrire et gérer les cours, les étudiants, les professeurs et les membres de l'administration

Fonctionnalités requises

  1. Une session membre de l'administration (Admin) disposant de super privilèges pour créer des cours, des emplois du temps...
  2. Une session professeurs avec des privilèges permettant de gérer leurs élèves.
  3. Gestion des inscriptions à l'université et aux cursus des élèves.

Mon travail

Résumé

Dans ce court projet mon rôle a été de réaliser la fonctionnalité d'emploi du temps pour les professeurs et étudiants. L'emploi du temps que j'ai codé se présente sous la forme d'un calendrier où dans chaque date s'affiche les cours de la journée et où il est possible en cliquant d'avoir une vue plus précise sur la journée.

Utilisation d'une base de données relationnelle

Avant ce projet je ne savais pas du tout comment utiliser une base de donnée dans un projet. Il s'est avéré que l'implémentation de cette fonctionnalité necessitait une grande compréhension des bases de données puisque il a fallut être capable d'afficher dans le calendrier uniquement les cours de la personne connecté. Pour cela il faut vérifier par exemple le rôle de l'utilisateur, ainsi que les groupes de travail dans lequel il est inscrit et enfin les cours reliés à ces groupes de travail.

Apprentissage de javascript et de cshtml

Ce projet m'a aussi permis d'obtenir un bon niveau en javascript car le calendrier a demandé beaucoup de travail afin d'avoir un résultat interactif et fonctionnel. Le cshtml a aussi été une découverte, il s'agit du langage html auquel on peut venir greffer des éléments de langage c# comme des variables ou des boucles.
