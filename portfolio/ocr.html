Logiciel de reconnaissance optique de caractères (OCR)

Rapport de projet
Etat : fini
Contexte : scolaire (EPITA, Villejuif)
Projet réalisé en groupe de 4.

Logiciel programmé en C utilisant un réseau de neurones permettant la reconaissance des caractères dans un texte écrit ou manuscrit pris en photo.

Outils informatiques utilisés

  • Langage C verison C99.
  • Compilateur gcc.
  • VIM en tant qu'environnement de développement.
  • Git pour collaborer et mettre à jour le code du projet.
  • Discord pour communiquer au sein de l'équipe.
  • Trello pour se répartir les tâches.

Cahier des charges du projet

Objectif global, le logiciel doit :

  • Prendre en entrée une image en entrée dans un format standard et produira un texte reconnu.
  • Disposer d'une interface graphique permettant de charger l'image, de la visualiser et sauver le texte produit.
  • Disposer d'une section apprentissage permettant d'entrainer un réseau de neurones et de sauver et/ou recharger le résultat de cet apprentissage.

Description détaillée du flux de traitement du logiciel :

  1. chargement de l’image
  2. suppression des couleurs (niveau de gris, puis noir et blanc)
  3. pré-traitement
  4. détection des blocs de textes
  5. détection des caractères
  6. reconnaissance des caractères détectés
  7. reconstruction du texte

Contraintes techniques du projet

  • Langage C version C99.
  • Utilisation de gcc pour compiler.
  • Le code doit compiler, sans warning avec les options : -Wall -Wextra -std=c99
  • Le projet est livré avec un fichier Makefile.
  • Tous les identifiants utilisés dans le code ainsi que les commentaires sont en anglais.
  • Dans le code les lignes ne doivent pas dépasser 80 colonnes et contenir d'espaces inutiles en fin de ligne.

Mon travail

Résumé

Dans ce projet j'ai eu un role très important dans la réalisation du code C permettant la préparation (en enlevant le "bruit") et la segmentation de l'image d'entrée qui contient du texte afin de la préparer à la reconaissance par le réseau de neurones. J'ai eu aussi un role dans la réalisation du réseau de neurones permettant la reconaissance des caractères et son implémentation en C.

Débruitage de l'image d'entrée

J'ai réalisé le code permettant de débruiter l'image à l'aide d'une technique de convolution cela consiste pour chaque pixel de l'image à déterminer sa valeur "débruitée" en utilisant la valeur des pixels avoisinants. Je ne connaissais pas ce principe avant ce projet et il s'agit d'une technique assez fondamentale en analyse d'image.

Processus de segmentation des caractères contenus dans l'image.

Le code réalisé dans cette partie a pour objectif de "découper" les caractères présent dans l'image. Cela se passe en 3 étapes, il faut d'abord isoler les lignes de textes en appliquant un premier marqueur sur les pixels entourant les lignes. Dans un second temps il faut répéter le processus pour isoler les caractères cette fois-ci en appliquant un nouveau marqueur sur leur bordure. La dernière étape la plus compliquée techniquement est de stocker les caractères dans des bitmaps (image composée uniquement de pixel unicolor 0 ou 1) afin de pouvoir ensuite les envoyés dans le réseau de neurones pour reconaissance.

Réalisation du réseau de neurones

La compréhension de ce que est un réseau de neurones a été une tâche très difficile tellement mes connaissances de base sur le fonctionnement étaient limitées. Il faut savoir aussi que au commencement du projet je ne connaissais pas du tout le langage C. Mais à force de travail et de recherches j'ai pu réaliser ce qui a servit de modèle à un de mes collabarateurs pour implémenter un réseau de neurones permettant la reconaissance de caractère en C. Ce modèle est implémentable dans tous les langages puisqu'il s'agit d'une définition dite "abstraite" d'un réseau de neurones.
